## 리다이렉션과 파이프  

### 리다이렉션(redirection)  
- 표준 스트림 흐름을 바꿔줄 수 있다
- <, >을 사용한다
- 주로 표준 출력을 화면이 아닌 파일에 쓸 때 이용한다
  ```
  ls > test.txt
  ```
- 기존에 있던 내용 뒤에 append하고 싶을 때는 >>

### 파이프(pipe)
- 두 프로세스 사이에서 한 프로세스의 출력을 다른 프로세스의 입력으로 사용할 떄
- |을 사용한다
  ```
  ls -al | grep bash

  ps aux | grep bash
  ```
